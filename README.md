# Aliases
Aliases and Functions files

#### ***Installation.***

```bash
URL="https://raw.githubusercontent.com/dontdoxxmeplz/aliases/main/" && wget $URL/aliases -O ~/.bash_aliases && wget $URL/extract.sh -O ~/.extract.sh && . ~/.bashrc
```
